# SJCC1
Gerador de Layouts de Vídeo para Páginas HTML

- Possui frontend de teste na pasta public (usar o index.html)
- As rotas estão em routes/index.js
- É necessária uma chave de API para a API Youtube Data Api V3 para o servidor funcionar.
- O frontend de teste é ativado usando o Live Server do Visual estudio code com a porta local http://127.0.0.1:5500/
